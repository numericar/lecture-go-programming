การสร้างตัวแปรในภาษา Go จะใช้ var

โดยทุก ๆ ข้อมูลจะมีชนิดของมันเอง
โโยจะมีชนิดข้อมูลดังนี้
    - int       จัดเก็บตัวเลขโดยไม่มีทศนิยม โดยเก็บได้ทั้งจำนวนเต็มบวกและลบ
    - float64   จัดเก็บตัวเลขพร้อมทศนิยม
    - string    จัดเก็บข้อมูล
    - bool      จัดเก็บสถานะ จริง หรือ เท็จ
    - uint      จัดเก็บเลขจำนวนเต็ม แต่เก็บเฉพาะค่าบวก
    - int32     จัดเก็บเลขจำนวนเต็ม ขนาด 32 bit
    - rune      คล้ายกัย int32 แต่ใช้จัดเก็บอักขระ unicode
    - uint32    จัดเก็บเลขจำนวนเต็ม ขนาด 32 bit โดยเก็บเฉพาะจำนวนเต็มบวก
    - int64     จัดเก็บเลขจำนวนเต็ม ขนาด 64 bit

null ของภาษา go แต่ละชนิดข้อมูล
    - int: 0
    - float64: 0.0
    - string: ""
    - bool: false
